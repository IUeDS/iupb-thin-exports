Stable tag: 0.1

== Description ==

Export Pressbooks books into a Thin Common Cartridge. Supports making both inline imsmanifests
and zips. Can export CC 1.2, 1.3, and thin.

== Installation ==

1. Upload `iupb-thin-exports/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to *wp-admin/tools.php?page=iupb-thin-export.php
